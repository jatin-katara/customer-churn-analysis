# Customer Churn Analysis

This project analyzes customer churn behavior using Python.

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib

## Project Overview
The dataset contains 64k+ customer records.  
Exploratory Data Analysis (EDA) was performed to identify churn patterns.

## Key Insights
- Customer churn distribution
- Churn by gender
- Churn by tenure
- Churn by subscription type
